Definición de Límite

Cálculo Diferencial - Licenciatura

Concepto Intuitivo

En matemáticas, el concepto de límite describe el comportamiento de una función cuando su argumento se acerca a un cierto punto. Decimos que el límite de $f(x)$ cuando $x$ tiende a $a$ es $L$, y escribimos:

$$\lim_{x \to a} f(x) = L$$

si para todo $\epsilon > 0$ existe un $\delta > 0$ tal que si $0 < |x - a| < \delta$, entonces $|f(x) - L| < \epsilon$.

Definición Formal (épsilon-δ)

Sea $f$ una función definida en algún intervalo abierto que contiene al punto $a$ (excepto posiblemente en $a$ mismo). Decimos que el límite de $f(x)$ cuando $x$ tiende a $a$ es $L$ si:

$$\forall \epsilon > 0, \exists \delta > 0 : 0 < |x - a| < \delta \implies |f(x) - L| < \epsilon$$

Esta es la conocida definición épsilon-delta del límite.

Ejemplos de Límites

Ejemplo 1: Límite de una función lineal

Calcular el límite de $f(x) = 3x + 2$ cuando $x$ tiende a 4:

$$\lim_{x \to 4} (3x + 2) = 3(4) + 2 = 14$$

Esto significa que cuando $x$ se acerca a 4, $f(x)$ se acerca a 14.

Ejemplo 2: Límite de una función racional

Calcular el límite:

$$\lim_{x \to 2} \frac{x^2 - 4}{x - 2}$$

Solución: Factorizamos el numerador:

$$\frac{x^2 - 4}{x - 2} = \frac{(x-2)(x+2)}{x-2} = x+2 \quad \text{para } x \neq 2$$

Por lo tanto:

$$\lim_{x \to 2} \frac{x^2 - 4}{x - 2} = \lim_{x \to 2} (x+2) = 4$$

Propiedades de los Límites

Si $\lim_{x \to a} f(x) = L$ y $\lim_{x \to a} g(x) = M$, entonces:

  • $\lim_{x \to a} [f(x) + g(x)] = L + M$
  • $\lim_{x \to a} [f(x) - g(x)] = L - M$
  • $\lim_{x \to a} [f(x) \cdot g(x)] = L \cdot M$
  • $\lim_{x \to a} \frac{f(x)}{g(x)} = \frac{L}{M}$, si $M \neq 0$
  • $\lim_{x \to a} [c \cdot f(x)] = c \cdot L$, donde $c$ es una constante
  • $\lim_{x \to a} [f(x)]^n = L^n$, para cualquier entero $n$

Límites Especiales

$$\lim_{x \to 0} \frac{\sin x}{x} = 1$$
$$\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e$$
$$\lim_{x \to 0} \frac{e^x - 1}{x} = 1$$
$$\lim_{x \to 0} \frac{\ln(1 + x)}{x} = 1$$
